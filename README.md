Programa que permite gestionar un cine con diferentes películas, salas y horarios, y calcular el precio según el tipo de proyección.
Características:
  * 4 películas disponibles (2 clásicas y 2 en 3D)
  * Salas independientes para cada película
  * 2 sesiones por día (Mañana y Tarde)
  * 50 butacas por sala
  * Cálculo automático de precios con recargo para películas 3D
  * Excepción personalizada que se lanza cuando se intenta reservar un asiento ya ocupado.
 
 **Funcionalidades
Gestión de Cartelera
  * Visualización de películas disponibles organizadas por tipo (Clásica/3D)
  * Selección interactiva de película
  * Información de precios por entrada

Sistema de Reservas
  * Selección de horario (Mañana/Tarde)
  * Visualización de disponibilidad de asientos en tiempo real
  * Reserva de múltiples asientos en una misma sesión
  * Validación de asientos ocupados
  * Opción de realizar múltiples reservas

Cálculo de Precios
  * Películas Clásicas: Precio base de 8.40€
  * Películas 3D: Precio base + 30% de recargo (10.92€)
  * Cálculo automático del precio total según número de entradas
